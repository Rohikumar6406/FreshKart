FreshKart is a modern and responsive grocery shopping frontend application built to solve the problem of cluttered and outdated grocery shopping UIs. The application delivers clean navigation, well-structured categories, smooth responsiveness, and a user-friendly interface using reusable, modular components.

@ Features

Clean and intuitive navigation
Well-structured grocery categories
Fully responsive design
Reusable and modular React components
Modern UI with Tailwind CSS
Smooth sliders using Swiper.js
Client-side routing with React Router

@ Tech Stack

React
JavaScript
Tailwind CSS
React Router
React Icons
Swiper.js

git clone https://github.com/Rohikumar6406/FreshKart.git
cd FreshKart
npm install
